Literature Survey

In today's digital era, web programming proficiency is essential for aspiring developers. To meet the escalating demand, educators and researchers are crafting effective learning environments. This literature review unveils pivotal insights, driving the creation of Web Block Craft, an innovative platform tailored for novice web programmers. Challenges in integrating web technologies into curricula inspire Web Block Craft's standalone, interactive approach, while success with Blockly-based education underscores its focus on computational thinking. Emphasizing accessible learning experiences, the platform's integration of visual blocks aligns with advocacy for playful introductions to programming.

Research Gap

Despite the increasing importance of web programming skills, there exists a gap in educational tools that effectively introduce web programming to children as well as beginners. While coding skills are crucial in today's digital economy, traditional code editors often present complexities that hinder effective learning, especially for novices. Existing educational platforms may lack the intuitive and visually driven approach necessary to engage younger learners and beginners effectively. Moreover, there is a scarcity of comprehensive learning tools that cover both frontend and backend web development, including key concepts such as HTML, CSS, JavaScript, DOM manipulation, server creation, web app security, APIs, and database handling.

Current educational resources may not adequately address the need for structured learning experiences with step-by-step guides tailored for beginners. Furthermore, there is a gap in integrating custom blocks specifically designed for web programming concepts within a flexible framework, such as Google Blockly. These custom blocks could enhance the practical understanding of both frontend and backend development, enabling users to transition seamlessly from visual representation to code implementation.

Research Problem and Solution

Problem

Current web development learning lacks engaging interfaces and struggles to keep beginners interested. Traditional code editors are complex, and few platforms cover both front-end and back-end aspects. Structured learning experiences for beginners are scarce, and underutilized tools like Google Blockly could make learning more practical.

Solution

Web Block Craft uses Google Blockly to create a fun and visual way to learn web development. Drag-and-drop blocks simplify coding concepts for beginners, covering both front-end and back-end aspects like HTML, CSS, and databases. With step-by-step guides, users can confidently build web projects, bridging the digital divide and equipping them with essential coding skills.

Research Objectives

Objective 01

To design an intuitive and visually driven educational platform for web programming, specifically targeting children aged eleven and above, as well as beginners.

Objective 02

To address the complexity barrier in traditional code editors by leveraging the Google Blockly framework and integrating custom blocks tailored to web programming concepts.

Objective 03

To provide comprehensive coverage of both frontend and backend web development concepts, including HTML, CSS, JavaScript, server creation, web app security, APIs, and database handling.

Objective 04

To offer structured learning experiences with step-by-step guides tailored for beginners, enhancing understanding and practical application of web programming concepts.

Objective 05

To empower users to develop web projects with confidence and creativity, bridging the digital divide and equipping the younger generation with essential coding skills for the future.

Frontend Builder

The Frontend Builder of Web Block Craft, is where you can bring your web design ideas to life. With our intuitive interface, you can experiment with HTML, CSS, and JS in real-time. Watch your code come alive as you make changes, and enjoy the visual representation to see exactly what's happening on your webpage. Let your creativity soar as you craft stunning designs effortlessly!

Backend Builder

The Backend Builder of Web Block Craft, is where you can turn your ideas into robust and powerful web applications. With our user-friendly interface, you can effortlessly create servers, handle APIs, manage databases, and implement authentication and authorization features. Experience the magic of backend coding as you build the backbone of your web projects, and watch your creations come to life with ease!
